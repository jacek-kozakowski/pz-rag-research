# API Manual
This file lists all API keys to put in `.env` file and where to get 
them from.

## [OpenAI](https://platform.openai.com) 
Better embedding models and summarization.
Costs money, not necessary, can be replaced with Groq.

## [Groq](https://console.groq.com/keys)
Can replace OpenAI as LLM model.

## [Tavily](https://app.tavily.com/home)
1000 free requests. When maxed out there is a fallback to DuckDuckGo
